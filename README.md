# GJQu.github.io

Personal portfolio and blog built with [Jekyll](https://jekyllrb.com/) using the [al-folio](https://github.com/alshedivat/al-folio) theme.

## Sections

- **About** — Bio, research interests, latest posts
- **Blog** — Posts on economics, data science, and everyday life
- **Projects** — Side projects in data science and web development
- **CV** — Embedded PDF resume

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Site runs at `http://localhost:4000`. See [CUSTOMIZATION.md](CUSTOMIZATION.md) for theme configuration details.

## Deployment

Pushes to `main` auto-deploy via GitHub Actions to [GJQu.github.io](https://GJQu.github.io).
